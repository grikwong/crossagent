# Code of Conduct

## Our Standard

This project expects respectful, professional collaboration. Participants should:

- Assume good faith and discuss technical disagreements directly.
- Focus on behavior, code, and outcomes rather than personal attacks.
- Be constructive in review, issue discussion, and design debate.
- Respect the time and boundaries of maintainers and contributors.

Unacceptable behavior includes harassment, discrimination, intimidation, doxxing, and sustained disruptive conduct.

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and other public communication channels for Crossagent.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct. Maintainers may also temporarily or permanently restrict participation as needed to protect the project.

## Reporting

Report conduct issues privately to the maintainer listed in [`CONTRIBUTORS.md`](./CONTRIBUTORS.md). Include links, screenshots, or other context when possible.
